Accessible Word Grid

Keyboard Shortcuts

  • 1 = Start Game
  • 2 = Read Letters
  • Space = Hold to Talk
  • Enter = Submit Typed Word

Voice Commands

  • Guess [word] = Submit a word
  • Hint = Get a hint
  • Read grid = Hear the grid letters
  • Repeat = Repeat the last status message
  • Read found words = Hear found words
  • Read remaining words = Hear remaining words
