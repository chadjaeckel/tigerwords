    Keyboard Shortcuts

    • 1 = Start Game
    • 2 = Read Letters
    • Space = Hold to Talk
    • Enter = Submit Typed Word

    Voice Commands

    • Guess [word] = Submit a word
    • Hint = Get a hint
    • Read grid = Hear the grid letters
    • Repeat = Repeat the last status message
    • Read found words = Hear the words already found
    • Read remaining words = Hear a summary of words left
